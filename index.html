GenChal 2022: Feedback Comment Generation for Writing Learning

Task overview

Feedback comment generation is a task where, given an input text, a system generates hints or explanatory notes that help language learners improve their writing skills.

For more details of the task, refer to the task definition page and our task proposal paper.

Please register from here in order to participate in the task. Since we'll be evaluating the submissions manually, we'll limit the total number of participating systems at around 10.

We closed the registration because we reached full capacity. Thank you for your interests!

The evaluation results are out! They can be viewed on the results page.
